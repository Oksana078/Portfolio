ShoeZone - assistant in shoe selection.

ShoeZone is an online shoe store for all ages and styles. Users can choose from different categories (men's, women's, children's shoes) and filter products by price, size, brand, and shoe type. Each product has a detailed description, photo, available sizes, and price. Users can add items to the cart, edit the quantity, or delete them. Main features:

  • Main menu page: categories of dishes (for example, appetizers, main courses, desserts, drinks).
  • The dish page: each dish has a name, description, photo, price, and an option to add to cart.
  • Cart: the ability to add items to the cart, edit the quantity, and delete items.
  • Checkout: a form for entering data and confirming the order.
